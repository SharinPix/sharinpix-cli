# Usage
<!-- usage -->
```sh-session
$ npm install -g sharinpix
$ sharinpix COMMAND
running command...
$ sharinpix (-v|--version|version)
sharinpix/0.0.0 linux-x64 node-v10.10.0
$ sharinpix --help [COMMAND]
USAGE
  $ sharinpix COMMAND
...
```
